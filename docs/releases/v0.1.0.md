# v0.1.0

Initial public template release.

## Highlights

- private password hashing template for Cloudflare Workers
- `WorkerEntrypoint` RPC contract for `hashPassword()` and `verifyPassword()`
- canonical presets `standard-2026q1` and `free-tier-fallback-2026q1`
- Better Auth migration-friendly helpers and examples
- benchmark history preserved on `codex/benchmark-workspace`

## Notes

- `standard-2026q1` is the repository's OWASP-aligned floor
- `free-tier-fallback-2026q1` exists as an operational fallback for constrained Workers Free deployments
- use `verifyAndMaybeRehash()` when upgrading stored hashes after successful login
