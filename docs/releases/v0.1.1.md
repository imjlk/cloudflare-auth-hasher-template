# v0.1.1

Patch release that brings the public release snapshot in line with the current `main` template.

## Highlights

- safer `verifyAndMaybeRehash()` persistence flow without passing plaintext passwords to callbacks
- safer example Workers that no longer return replacement hashes over HTTP
- richer `GET /` metadata with algorithm, template version, and artifact checksum
- stronger parity and generated-type validation for the root Worker and example Workers

## Notes

- this release keeps the same template positioning and canonical presets from `v0.1.0`
- `standard-2026q1` remains the repository's OWASP-aligned floor
- `free-tier-fallback-2026q1` remains an operational fallback, not the security baseline
